#  Sales Taxes
### Build the project

```sh
cd src/main/java/com/itemis/conftrackmanagement
javac -d classes -cp classes  *.java ./session/*.java ./talk/*.java ./track/*.java
```


### Run the project
```sh
java -cp classes com.itemis.conftrackmanagement.Main
```


### Run tests
```sh
mvn test
```
